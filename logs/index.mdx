## What are Logs?

Logs in EnSync provide a detailed record of all activity in your workspace, including events, apps, integrations, and system operations. They help you monitor system health, troubleshoot issues, and maintain an audit trail for compliance purposes.

## Log Types

EnSync logs are categorized into several types to help you quickly find the information you need:

### Event Logs

Records of event publishing and delivery:
- When events are published
- Which apps or integrators published them
- When events are delivered to subscribers
- Delivery failures and reasons

### App Logs

Activity related to your internal apps:
- App creation and configuration
- Permission changes
- Key regeneration
- Connection status changes

### Integration Logs

Activity related to integration pages and partners:
- Integration page creation and updates
- Partner connections and disconnections
- Partner subscription changes
- Partner authentication events

### System Logs

Platform-level operations and status:
- System maintenance
- Service status changes
- Configuration updates
- Performance metrics

## Using the Logs Interface

The Logs section provides powerful tools for monitoring and troubleshooting:

### Filtering Logs

Use the filter controls to narrow down logs:
- **Type**: Filter by Event, App, Integration, or System logs
- **Level**: Show only Info, Warning, Error, or Critical logs
- **Source**: Filter by specific apps, integrations, or system components
- **Time Range**: Set a custom date and time range
- **Search**: Find logs containing specific text

### Viewing Log Details

Click on any log entry to expand it and see full details:
- Complete message text
- Technical details in JSON format
- Related entities (apps, events, integrations)
- Stack traces for errors

### Exporting Logs

Export logs for offline analysis or record-keeping:
- Click the "Export" button to download filtered logs
- Choose CSV or JSON format
- Set a date range for the export

## Accessing Logs

Access the Logs section in the EnSync Dashboard to:
- View real-time activity across your workspace
- Filter and search logs by type, level, source, and time range
- Export logs for offline analysis or compliance
- Monitor system health and troubleshoot issues

The dashboard provides an intuitive interface for all log management tasks.
